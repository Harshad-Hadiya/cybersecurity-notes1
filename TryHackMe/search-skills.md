## Search Engines
Google is the most commonly used search engine
Other tools can also help find technical and security-related information

## Search Techniques

# Using quotes (" ")
Search for exact phrases.
Example:
"what is networking"

# Using site: operator
Search within a specific website.
Example:
site:tryhackme.com networking

# Using filetype:
Search for specific file types.
Example:
filetype:pdf cybersecurity basics

# Using - (minus)
Exclude unwanted results.
Example:
linux commands -windows
