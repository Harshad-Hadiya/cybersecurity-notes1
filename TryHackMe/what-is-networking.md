##  What is a Network?
A network is a group of devices connected together to share data and resources.

## Types of Networks
# LAN (Local Area Network)
A small network within a limited area like home or office.
#  WAN (Wide Area Network)
A large network that connects multiple LANs, such as the Internet.

## Internet vs Intranet
Internet = Global public network
Intranet = Private internal network

 ## IP Address
A unique address assigned to each device on a network used for identification and communication.

## MAC Address
A physical address assigned to a network device by the manufacturer.

## Protocols
Rules that define how data is transmitted over a network.

## Common Protocols
HTTP → Web browsing
HTTPS → Secure web browsing
FTP → File transfer
DNS → Domain name to IP conversion

## Key Learning
Networking is the foundation of cybersecurity because all systems communicate over networks.
